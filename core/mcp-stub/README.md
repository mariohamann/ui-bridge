# ui-bridge-mcp

This is a placeholder package. The **UI Bridge MCP server** must be installed locally in your project as [`@ui-bridge/mcp`](https://www.npmjs.com/package/@ui-bridge/mcp).

To run it, use the `--no` flag so `npx` resolves the local binary instead of fetching from the registry:

```json
{
  "command": "npx",
  "args": ["--no", "ui-bridge-mcp"]
}
```
